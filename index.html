DDD Brisbane

DDD Brisbane is a non profit community event run by developers for developers. Anyone can submit a session to the event and delegates then vote on the sessions they want to see.

Last year's event sold out with over 230 attendees, and the event has been growing steadily year on year.

We have various levels of sponsorship available for 2015 which include logo placement on the website, agenda, delegate slides, and speaker shirts as well as advertisements in delegate packs and a presence on the day.

Brisbane .NET User Group

Formerly the QMSDN User Group, this is the new face of the community group focused around software development in the Microsoft space. This includes but is not limited to the .Net platform, and encompasses not only Microsoft products and techniques, but also open source and other alternatives.

The Brisbane .NET User Group has more than 800 members and averages approximately 80 attendees each month.
